## Problem-Solving Approach

For each problem, I aim to practice:

1. understanding the input and output
2. identifying the pattern
3. writing a brute-force solution first
4. improving it to a more efficient solution
5. analyzing time and space complexity
6. rewriting until the code is clean and readable
